# Financial Analysis of Tech Companies (2010–2022)

This project analyzes the financial performance of major technology companies (AAPL, AMZN, GOOG, MSFT, NVDA) over the period 2010–2022.

The analysis focuses on:
- Revenue Growth
- Profit Margin
- EBITDA Margin

The goal is to identify differences in growth patterns, profitability, and business strategies across companies.

## Files
- financial_analysis.xlsx → data cleaning and analysis
- financial_analysis_presentation.pdf → visual insights and conclusions
